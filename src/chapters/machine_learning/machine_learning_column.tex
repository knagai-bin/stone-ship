\begin{column}{ミニ研究}
\textbf{深層学習による交通流予測モデルの検討}
{\small 李康民}

\subsubsection*{研究の背景と目的}

本研究では，高速道路の交通流データを用いて，将来の交通状態を予測するモデルを検討した．
目的は大きく二つである．第一に，単純なNeural NetworkであるMulti-Layer Perceptron（MLP）を用いて，交通量$Q$，速度$V$，交通密度$K$の短期予測を行うことである．
第二に，MLPをMoving AverageおよびLWR方程式に基づくモデルと比較し，どのようなデータ条件でNeural Networkが有効に働くのかを確認することである．

ここで特に注目したのは，NNの性能を高めるためにはモデル構造だけでなく，入力データの性質や前処理がどの程度重要になるのか，という点である．

\subsubsection*{使用データと予測タスクの設定}

使用したデータは，2024年1月の東名高速道路における5分間隔の交通流データである．
データには，時刻，KP，lane，交通量$Q$，速度$V$，交通密度$K$が含まれている．
まず時空間分布を確認すると，交通量$Q$には明確な日周期性が見られた．
夜間には交通量が低下し，日中に再び増加するというパターンが繰り返されている．
一方，速度$V$は多くの時間帯で比較的安定しており，全体としては周期性と短期的安定性が強いデータであると考えられる．

\begin{center}
\begin{minipage}{0.80\linewidth}
\centering
\includegraphics[width=\linewidth]{assets/machine_learning/traffic_heatmap.png}
\captionof{figure}{交通量$Q$と速度$V$の時空間分布}
\label{fig:ml-traffic-heatmap}
\end{minipage}
\end{center}

laneごとの集計では，交通量や速度の絶対値に大きな差が見られた．
特に一部のlaneでは交通量のピークが大きく，一方で別のlaneでは交通量が小さく，欠損率も高かった．
欠損率の高いlaneをそのまま学習に利用すると，モデルが本来の交通状態ではなく，データの欠損構造を学習してしまう恐れがある．

予測タスクとしては，Lookbackを12，Horizonを12に設定し，過去60分のデータから未来60分の交通状態を予測する問題を基本ケースとした．
データはTrain，Validation，Testに0.6，0.2，0.2の割合で分割し，Epochは12回としてMLPを学習した．

上記のBaseline条件における予測性能比較より，Moving Averageが最も高い性能を示した．
特に$Q$と$V$のRMSE，MAEはMLPよりも小さく，Congestion F1も高い値となった．
これは，今回の交通流データでは「現在の交通状態がしばらく継続する」という単純な仮定が非常に有効だったことを意味する．
交通量には日周期性が強く，速度も短時間では大きく変化しにくいため，Moving Averageのような単純なbaselineでも十分に高い性能を示したと考えられる．

一方で，MLPは基本的な周期パターンを学習できているものの，Moving Averageを上回ることはできなかった．
LWRモデルは交通流理論に基づくモデルであるが，今回のデータでは欠損や観測点ごとのばらつきの影響を受け，誤差が大きくなった．

\subsubsection*{MLPによる予測結果の特徴}

\begin{center}
\begin{minipage}{0.70\linewidth}

MLPのLearning Curveを見ると，Epoch 0から2付近でlossが急激に低下している．
これは，MLPが交通データに含まれる基本的な周期性や平均的な傾向を早い段階で学習していることを示している．
一方で，その後のvalidation lossの改善は小さく，学習を続けても大きな性能向上は見られなかった．

\begin{center}
\begin{minipage}{0.70\linewidth}
\centering
\includegraphics[width=\linewidth]{assets/machine_learning/mlp_prediction.png}
\captionof{figure}{Simple MLPによる交通量$Q$の予測}
\label{fig:mlp-prediction}

また，予測例を見ると，MLPの予測値は実測値の細かな上下変動を追いかけるというよりも，平均的な値に近づく傾向があった．
これは，MSEを最小化する学習では，不確実な変動を無理に予測するよりも，平均的な値を出す方が損失を小さくしやすいためである．
特にMSEを損失関数として用いた場合，予測が困難な急激変動を無理に追従するよりも，平均的な値へ収束する方が損失を安定して小さくできるため，結果として平坦な予測が生成されやすい．
今回のMLPは安定して学習していたが，突発的な変動や局所的な非線形パターンを十分に再現するには限界があった．

\subsubsection*{欠損率の高いlaneを除外した場合}

次に，欠損率の高いlaneを除外して同じ予測タスクを行った．
これは，欠損の多いデータがMLPの学習にノイズとして作用しているかを確認するためである．

欠損率の高いlaneを除外した場合の予測性能比較を見ると，Simple MLPの$Q$に対するRMSEとMAEがBaselineより改善している．
これは，欠損率の高いlaneを除くことで，MLPがより安定した交通パターンを学習できるようになったためである．
一方で，Congestion F1ではMoving Averageの方が高く，渋滞状態の判定では単純な平均ベースの手法が依然として強い結果となった．
この結果から，NNの性能改善には，モデル構造の変更だけでなく，入力データの品質管理が重要であることが分かる．

\subsubsection*{Lookbackの変更と時間依存性}

さらに，Lookbackを12から6に短くし，過去30分のデータから未来60分を予測する設定を試した．
これは，MLPが長い過去履歴を利用しているのか，それとも直近の交通状態に主に依存しているのかを確認するためである．

Lookbackを6に変更した場合の予測性能比較を見ると，MLPの$Q$に対する誤差は大きく悪化せず，むしろBaselineより改善した．
このことから，今回のMLPは長い過去履歴よりも，直近の交通状態や時間帯ごとの平均的なパターンに強く依存して予測していたと考えられる．

ただし，$K$のRMSEは不安定であり，交通密度の予測には外れ値やデータのばらつきが強く影響していた可能性がある．

また，Lookbackを短くした背景には，入力履歴を過度に長くすると，MLPが局所的なノイズや短期変動まで過学習してしまう可能性があると考えたためである．
一方で，今回の結果ではLookbackを変更しても性能差は限定的であり，現在のMLPは長期履歴よりも直近状態へ強く依存している可能性が示唆された．
なお，交通流の周期性をより明示的に学習させるためには，24時間単位などより長いLookbackを設定する方法も考えられるが，計算量やメモリ使用量の増加が課題となる．
また今回は実装できなかったが，単純に直前時刻の系列だけを入力するのではなく，同一曜日・同一時間帯における過去データを特徴量として利用する方法も，交通流の周期性を捉える上で有効であると考えられる．

\subsubsection*{特徴量重要度とMLPの学習傾向}

Moving Averageは，直近の交通状態をそのまま利用する単純なモデルであるが，今回のデータでは高い予測性能を示した．
これは，交通流データに強い短期安定性と周期性が存在しており，「現在の状態がしばらく継続する」という仮定が有効であったためであると考えられる．
一方で，Moving Averageは急激な交通状態変化や局所的なピーク変動を表現することが難しく，予測結果が平滑化されやすいという特徴も確認された．

SHAPによる特徴量重要度を確認すると，速度$V$が最も重要な特徴量として利用されていた．
速度変化は混雑状態や流量変化を早く反映するため，短期予測において重要な情報になったと考えられる．
また，直近stepの特徴量が強く利用されており，MLPが主に短期的な交通状態に依存して予測していることが分かる．
これは，Lookbackを短くしても性能が大きく変化しなかった結果とも整合的である．

\subsubsection*{結びに}

本研究では，東名高速道路の交通流データを用いて，MLP，Moving Average，LWRモデルによる短期予測を比較した．
その結果，MLPは周期的な傾向を学習できるものの，予測値が平均的で平坦になりやすく，Moving Averageを明確に上回るには至らなかった．

また，Lookbackを短くしても性能が大きく変化しなかったことや，SHAPで直近stepの重要度が高かったことから，今回のMLPは長期依存関係よりも，直近の交通状態に強く依存して予測していたと考えられる．

また，欠損率の高いlaneを除外することでMLPの性能が改善したことから，NNの性能はモデル構造だけでなく，入力データの品質に大きく依存することが確認された．
今回のデータでは，交通状態そのものが比較的予測しやすく，複雑な非線形性が強く現れていなかった可能性がある．
本研究を通じて，交通流予測においては単純にモデルを複雑化するだけでは十分ではなく，交通データが持つ周期性，欠損構造，lane構造，短期安定性といった特性を理解した上で，適切な前処理や特徴設計を行うことが重要であることが確認された．
今後は，RNNやTransformerのような時系列モデルとの比較に加え，交通流の周期性やlane間の関係性をより適切に表現できる特徴設計についても検討する必要があると考えられる．
